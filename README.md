# 2048 Game

## Описание
2048 - это увлекательная и простая игра-головоломка, в которой игроку нужно соединять плитки с одинаковыми числами, чтобы достичь плитки с числом 2048. Игра написана на языке программирования C++ с использованием фреймворка Qt.

## Установка

### Требования
- C++ компилятор
- Qt 5.15 или выше
- Git

### Инструкции
1. Клонируйте репозиторий:
   ```sh
   git clone https://github.com/Evggeenn/game.git
   ```
2. Перейдите в директорию проекта:
   ```sh
   cd 2048
   ```
3. Откройте проект в Qt Creator или другой среде разработки, поддерживающей Qt.
4. Соберите проект и запустите его.

## Использование
После запуска игры вы увидите игровое поле размером 4x4, заполненное плитками с числами. Используйте стрелки на клавиатуре, чтобы перемещать плитки в выбранном направлении. Плитки с одинаковыми числами будут соединяться, образуя плитку с удвоенным числом. Цель игры - создать плитку с числом 2048.
